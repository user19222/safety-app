# 🏗️ Safety App - אפליקציית בטיחות באתרי בנייה

אפליקציה מקיפה לניהול בטיחות באתרי בנייה, עם תמיכה מלאה בעברית ואנגלית.

## 🌟 תכונות עיקריות

### 📊 לוח בקרה (Dashboard)
- מעקב אחר מספר עובדים פעילים
- ניטור מפגעי בטיחות פתוחים
- מדד בטיחות כללי
- גישה מהירה לכל התפקידים

### ⚠️ ניהול מפגעים
- **דיווח מפגע חדש**
  - תיאור מפורט
  - דירוג חומרה (חמור/בינוני/קל)
  - צילום מפגע
  - בחירת קטגוריית סיכון
  - מיקום מדויק
- **רשימת מפגעים**
  - הצגה ויזואלית ברורה
  - סינון לפי סטטוס
  - סימון מפגעים כטופלו
  - פרטים מלאים לכל מפגע

### 👷 ניהול עובדים
- רשימת עובדים באתר
- תפקידים ואחריות
- מידע ליצירת קשר
- מעקב נוכחות

### ✅ צ'ק-ליסט יומי
- בדיקות בטיחות יומיות
- קטגוריות מרובות:
  - ציוד מגן אישי (PPE)
  - גישה ואבטחת אתר
  - הגנה מפני נפילות
  - בטיחות חשמל
  - בטיחות אש
  - ניקיון ותחזוקה
- אפשרות להוספת הערות
- שמירה אוטומטית

### 📈 דוחות
- דוח שבועי/חודשי
- ניתוח מגמות
- סיכום מפגעים לפי קטגוריה
- סיכום לפי רמת חומרה
- ייצוא ל-PDF או Excel

### 🔔 התראות
- התראות בזמן אמת
- עדכונים על מפגעים חדשים
- סימון מפגעים כטופלו
- השלמת צ'ק-ליסט

## 🌐 תמיכה רב-לשונית
- **עברית** - ממשק מלא מימין לשמאל (RTL)
- **אנגלית** - ממשק משמאל לימין (LTR)
- מעבר חלק בין השפות

## 💾 אחסון מקומי
- שמירה אוטומטית ב-LocalStorage
- שימור נתונים בין הפעלות
- אין צורך בשרת

## 🎨 עיצוב
- עיצוב מודרני ונקי
- Tailwind CSS
- Responsive Design - מתאים לנייד וטאבלט
- אייקוני אימוג'י פשוטים וברורים
- צבעים המבדלים רמות חומרה

## 🚀 איך להריץ

1. **פתח את הקובץ בדפדפן**:
   ```bash
   # Windows
   start safety-app.html
   
   # macOS
   open safety-app.html
   
   # Linux
   xdg-open safety-app.html
   ```

2. **או גרור את הקובץ לדפדפן**

זהו! האפליקציה עובדת ישירות בדפדפן ללא צורך בהתקנה.

## 📱 התקנה כ-Progressive Web App (PWA)

בדפדפנים נתמכים, ניתן להתקין את האפליקציה על המכשיר:

1. פתח את האפליקציה בדפדפן
2. לחץ על תפריט הדפדפן
3. בחר "התקן אפליקציה" או "Add to Home Screen"
4. האפליקציה תופיע כאפליקציה עצמאית

## 🛠️ טכנולוגיות

- **React 18** - ספריית UI
- **Babel Standalone** - transpiling של JSX בדפדפן
- **Tailwind CSS** - עיצוב וסטיילינג
- **LocalStorage API** - אחסון נתונים מקומי
- **Emoji Icons** - אייקונים פשוטים ואוניברסליים

## 📋 שימוש באפליקציה

### דיווח מפגע חדש
1. לחץ על כפתור ➕ הצף בתחתית המסך
2. מלא תיאור מפורט
3. בחר רמת חומרה
4. צלם תמונה (אופציונלי)
5. בחר קטגוריית סיכון
6. הוסף מיקום
7. שלח דיווח

### צ'ק-ליסט יומי
1. עבור ללשונית "צ'ק-ליסט"
2. עבור על כל הקטגוריות
3. סמן עבר/נכשל/לא נבדק
4. הוסף הערות במקרה הצורך
5. שמור צ'ק-ליסט

### ניהול מפגעים
1. עבור ל"מפגעים פתוחים"
2. לחץ על מפגע לצפייה בפרטים
3. סמן כטופל כשנפתר
